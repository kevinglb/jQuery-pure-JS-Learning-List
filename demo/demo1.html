after() vs before() vs append() vs appendTo()

after(): Insert content, specified by the parameter, after each element in the set of matched elements.

before(): Insert content, specified by the parameter, before each element in the set of matched elements.

append(): Insert content, specified by the parameter, to the end of each element in the set of matched elements.

appendTo(): Insert every element in the set of matched elements to the end of the target.

Difference between after() and before()

It is clear that both functions add the parameter elements to the matched elements, the only difference is the position that the parameter of after() will be added after the matched elements while the parameter of before is added before the metched elements

Difference between after() and append()

That means the difference is that the after() will add the parameter after the target element, while the append() will add the parameter element inside the target element and as its last childElement

Difference between append() and appendTo()

The difference between both functions is that the target elememnts and the element to be added are used in opposite way. In append(), the parameter is the element to be added and the caller is the target element, but in appendTo(), the caller is the element to be added and the parameter is the target element. The order of the added element of these two functions depends on the functions called order.

$("div.outer").before("before");
$("div.outer").after("after");

The append() is called before appendTo(), so the result is that the parameter if append() is ahead of the appendTo()'s caller element
